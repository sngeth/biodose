# ⚡ BioDose

> Your daily dose of evidence-based health optimization

BioDose automatically fetches and displays peer-reviewed scientific articles focused on health optimization topics including sleep, cognitive performance, longevity, nutrition, and more.

## Features

- 🔬 **Peer-reviewed only** - All articles from trusted journals
- 🆓 **Open access** - Free full-text articles
- 🤖 **Automated daily** - New article every day at 7 AM
- 📱 **Mobile-friendly** - Responsive design
- ⚡ **Fast & lightweight** - Static site, loads instantly

## Topics Covered

- Sleep quality & circadian rhythm
- Cognitive performance & nootropics
- Longevity & anti-aging
- Metabolic health & nutrition
- Exercise science & recovery
- Stress management
- Supplements (evidence-based)

## How It Works

1. **Daily Fetch**: GitHub Actions runs `fetch-articles.js` every morning
2. **Smart Selection**: Searches PubMed for recent optimization-focused studies
3. **Scoring**: Ranks articles by relevance (RCTs, meta-analyses get priority)
4. **Auto-update**: Best article is published to the site automatically

## Data Sources

- PubMed Central (PMC)
- Open access peer-reviewed journals
- Free full-text only

## Setup & Deployment

### Option 1: GitHub Pages (Recommended)

1. **Fork this repository**

2. **Enable GitHub Pages**:
   - Go to Settings → Pages
   - Source: Deploy from branch `main`
   - Folder: `/` (root)
   - Save

3. **Your site will be live at**: `https://yourusername.github.io/biodose`

4. **Automatic updates**: The GitHub Action will run daily and update the site automatically!

### Option 2: Local Development

```bash
# Clone the repo
git clone https://github.com/yourusername/biodose.git
cd biodose

# Fetch an article
node fetch-articles.js

# Open index.html in a browser
open index.html
```

### Manual Article Fetch

To manually fetch a new article anytime:

```bash
node fetch-articles.js
```

This will update `latest-article.json` which the website reads.

## Customization

### Change Update Time

Edit `.github/workflows/daily-update.yml`:

```yaml
schedule:
  - cron: '0 7 * * *'  # Change to your preferred time (UTC)
```

### Modify Keywords

Edit `fetch-articles.js` and change the `KEYWORDS` array to focus on different topics.

### Style Changes

Edit `index.html` - all CSS is inline in the `<style>` tag.
